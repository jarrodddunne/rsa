## Synopsis

A simple python RSA encryption program. Times how long it takes to encrypt and decrypt with RSA keys of various bit lengths

## Motivation

I wanted to learn about RSA encryption, and time it on python vs various other languages.

## Installation

Just run "python3 rsa.py"
